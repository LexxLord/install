# install

**Install git**
```shell
pacman -Sy git glibc
```

**Clone repository**
```shell
git clone https://github.com/LexxLord/install.git /install
```

**Start install**
```shell
sh install-1
```

**Start configure**
```shell
sh install-2
```

**Configure SDDM (Autologin)**
```shell
sh install-3
```
