# install

**Install git**
```shell
pacman -Sy git glibc
```

**Clone repository**
```shell
git clone https://github.com/LexxLord/install.git /install
```

**Go to install**
```shell
cd /install
```

**Start installation**
```shell
sh install-1
```

**Continue installation**
```shell
cd /install
```

**Start configure**
```shell
sh install-2
```

**Configure SDDM (Autologin)**
```shell
sh install-3
```

**Set up a repository**
```shell
sh install-4
```
